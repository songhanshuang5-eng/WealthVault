# 💰 WealthVault 私人账本 v1.0

一款运行在本地电脑上的个人财务管理工具，数据存在自己电脑里，不上传任何服务器。

---

## 📦 下载安装

点击下方 **Assets → WealthVault.zip** 下载，解压后按系统操作：

### Mac 用户

1. 解压后，**双击 `安装App.command`**（首次运行一次即可）
2. 如果弹出"无法打开"，右键点击它 → 选择"打开" → 再点"打开"
3. 安装完成后，**双击 `WealthVault.app`** 启动
4. 可以把 `WealthVault.app` 拖到桌面或"应用程序"文件夹方便以后使用

### Windows 用户

1. 确认已安装 Python 3（没有的话前往 https://www.python.org/downloads/ 下载）
2. 解压后，**双击 `启动.bat`** 即可启动
3. 启动后浏览器会自动打开，若没有请手动访问 `https://localhost:8181`

---

## ✨ 功能一览

- **多账户管理** — 银行存款、各币种持仓、负债一目了然
- **投资持仓** — 基金、股票、固定收益产品，支持加仓/减仓/分红记录
- **货币兑换** — 记录换汇，自动扣减来源持仓余额
- **转账记录** — 账户之间资金流水
- **净资产总览** — 多币种汇总，历史快照走势图
- **手机同步** — 手机与电脑连同一 Wi-Fi，可手动推送/拉取数据
- **数据导出** — 一键导出 JSON 备份，随时可恢复

---

## 🔒 隐私说明

- 所有数据储存在**本地浏览器**中，不联网，不上传
- 应用有密码保护，首次使用时设置
- 导出的 JSON 文件由你自己保管

---

## ❓ 常见问题

**Q: 我的数据会和别人共享吗？**
不会。每个人下载安装后，数据完全独立存在各自的电脑上。

**Q: 关闭窗口后数据还在吗？**
在的。数据存在浏览器本地存储里，和网页历史一样持久。建议定期在"数据管理"页面导出 JSON 备份。

**Q: Mac 弹出"来自身份不明的开发者"怎么办？**
右键点击文件 → 选择"打开" → 再点"打开"，之后就不会再提示了。

**Q: 手机能用吗？**
可以。手机和电脑连同一个 Wi-Fi，在手机浏览器里输入电脑上显示的地址即可访问，支持添加到主屏幕作为 App 使用。
