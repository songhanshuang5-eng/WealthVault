WealthVault 用户手册

1. 安装与启动

WealthVault 运行在本地电脑上，数据完全存储在自己的设备中，无需注册账号、无需联网。Mac 用户可一键生成桌面应用，Windows 用户双击批处理文件即可启动。

Mac — 首次安装（只做一次）

1
前往 github.com/songhanshuang5-eng/WealthVault/releases，点击最新版本下方的 Source code (zip) 下载，解压得到 WealthVault 文件夹
2
双击 安装App.command
若弹出「Not Opened」或「无法打开」对话框，点 Done，然后：
系统设置 → 隐私与安全性 → 向下滚动找到被阻止的文件 → 点 「仍要打开」 → 再点 「打开」
此操作只需一次，是 macOS 对所有从网络下载脚本的正常安全检查。
3
脚本自动运行，完成后文件夹内出现 WealthVault.app
4
将 WealthVault.app 拖到桌面或「应用程序」文件夹
安装脚本会自动处理系统安全限制（Gatekeeper），此后双击 App 不会再弹出警告。

Mac — 日常启动

1
双击 WealthVault.app，图标在 Dock 中短暂跳动
2
约 2 秒后浏览器自动打开 https://localhost:8181
3
若浏览器显示「您的连接不是私密连接」（本地自签名证书的正常提示）：
Chrome：点页面空白处，键盘直接输入 thisisunsafe（无需回车）
Safari：点「显示详细信息」→「访问此网站」
此提示只在第一次出现，之后浏览器会记住。
4
进入登录页，输入账号密码开始使用
关闭 App（Cmd+Q）时服务器自动停止。若只关闭浏览器标签，服务器继续后台运行，下次直接在浏览器输入地址即可。

Windows — 启动方式

1
确认已安装 Python 3（python.org/downloads，安装时勾选「Add Python to PATH」）
2
双击 启动.bat，弹出黑色终端窗口
3
浏览器访问 https://localhost:8181，处理证书提示方法同上
终端窗口需保持开启，关闭终端即停止服务器。

手机安装为 App（PWA）

手机与电脑连同一 Wi-Fi，在手机浏览器中访问电脑的局域网地址（格式如 https://192.168.1.xxx:8181，启动时终端会显示具体地址）。

1
iOS Safari：点底部分享按钮 → 「添加到主屏幕」→「添加」
2
Android Chrome：点右上角菜单 → 「添加到主屏幕」或「安装应用」
3
桌面出现 WealthVault 图标，安装后离线也可使用，不依赖电脑
iOS 清除网站数据后数据会丢失，请定期导出 JSON 备份（见第 21 章）。

WealthVault 用户手册

WealthVault 是一款本地运行的个人财富管理工具，所有数据保存在您的浏览器本地，不上传到任何服务器。

主要功能

模块功能
🏦 账户多币种存款、负债、支出、定期收入计划
📈 投资固定收益 / 非固定收益持仓、仓位操作、分红、清仓、基准对比
💱 汇率货币兑换记录，自动更新账户余额
🔄 转账账户间资金划转
📊 总览净资产图表、资产分布、历史快照

2. 登录与安全

首次注册

1
访问应用地址，点击「新用户注册 →」
2
设置用户名（至少 2 个字符）和密码（至少 4 位）
3
注册成功后自动登录

日常登录

输入用户名和密码，点击「登 录」。登录成功后进入主界面，所有数字默认处于隐私模式（显示为 ***），点击右上角 🙈 可切换显示。

密码以 SHA-256 哈希形式存储在浏览器本地，无法被读取或恢复。如果忘记密码，只能通过「重置」功能清空数据后重新注册。

多用户

同一浏览器支持多个用户账号，每个用户的数据相互隔离。切换用户只需退出后以其他用户名登录。

17. 货币兑换

记录银行换汇操作，系统自动从指定持仓中扣减卖出货币并增加买入货币，精确控制资金来源。

新建兑换

1
选择兑换账户
2
选择卖出货币 + 买入货币
3
选择来源持仓（新功能）：系统列出该账户下所有该币种的正余额持仓
  • 只有一笔持仓时：自动选中，显示「已自动选中」提示
  • 有多笔持仓时：下拉菜单供手动选择（例如：活期 vs 某笔定期零头）
4
输入卖出金额和买入金额，系统自动显示汇率
5
点「💱 确认兑换」，来源持仓精确扣减，买入货币自动入账

选择来源持仓可防止换汇金额被错误扣减到定期存款或其他不应动用的持仓上。删除兑换记录时，金额会精确归还到原来的持仓。
余额校验基于所选持仓的单笔余额，超出时系统会阻止提交。

20. 多设备同步

手机与电脑连接同一 Wi-Fi 时，可通过「数据」页面手动推送或拉取数据，实现双向同步。

同步原理

电脑运行的服务器（server.py）充当中转站。推送时，本机数据上传到服务器；拉取时，另一台设备从服务器下载数据覆盖本机。

☁️ 多设备同步
📤 推送到服务器
把本机数据上传，另一台设备再拉取
推送本机数据
📥 从服务器拉取
覆盖本机数据为服务器上的版本
拉取最新数据

典型操作流程

电脑 → 手机：在电脑上点「推送本机数据」，再在手机上点「拉取最新数据」。

手机 → 电脑：在手机上点「推送本机数据」，再在电脑上点「拉取最新数据」。

拉取操作会覆盖本机当前数据，拉取前系统会弹出确认提示。如不确定，建议先导出备份。
同步仅在电脑运行服务器（WealthVault.app 或启动脚本处于运行状态）时有效。手机和电脑须连接同一局域网（同一 Wi-Fi）。

21. 数据备份与恢复

所有数据存储在浏览器 localStorage 中，换浏览器或清除缓存会导致数据丢失，请定期备份。

导出备份

1
点击顶部「💾 数据」Tab
2
点「下载 JSON 文件」，保存至本地或云端

导入恢复

1
点「选择 JSON 文件」，选择之前导出的备份文件
2
确认导入，当前数据会被覆盖
导入会覆盖现有所有数据，操作前请先导出一份备份。

备份提醒

每月首次打开 App 时，顶部会出现「📥 本月尚未备份数据」横幅提醒，点「立即备份」即可下载。

多账号数据隔离

不同用户名的数据互相独立，切换用户名登录后看到的是该用户的独立数据。

建议每月备份一次，文件名加上日期方便管理。迁移到新电脑时，导出后在新浏览器中登录并导入即可。
