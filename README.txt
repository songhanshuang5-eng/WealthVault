【启动方法】

▶ Mac / Linux：
  打开终端，进入本文件夹，运行：
    python3 server.py
  浏览器会自动打开。

▶ Windows：
  双击 "启动.bat" 文件，
  或打开命令提示符运行：
    python server.py

▶ 手机访问（需与电脑连同一 WiFi）：
  启动服务器后，终端会显示手机地址，
  例如：http://192.168.1.8:8080/
  在手机浏览器中输入该地址即可。

  iPhone 添加到主屏幕：
  Safari 打开后 → 分享按钮 → 添加到主屏幕

【数据说明】
  数据自动保存在浏览器本地（localStorage），
  关闭页面后依然保留。
  建议在"数据"页定期导出 JSON 备份。

【端口修改】
  如需更换端口，编辑 server.py 第一行：
    PORT = 8080  →  改为你想要的数字
